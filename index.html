gitx by cyberfox

A gitk clone for OS X

GitX is an easy-to-use Git facade for Mac OS X; I've modified it to include some rudimentary support for remote repositories. It also provides an interface to git for users who aren't inclined to use Terminal.app, but need access to a git repository. This is how I got involved with it, adding remote repository support so that several designer co-workers wouldn't have to learn the complexities of git's command-line operations. I primarily use it for the history/changes view.

Dependencies

git must be installed on the system and in the default path, as it calls the git executable under the hood.

Install

Uncompress the .zip file, and move the app to where you'd like it to run from, most likely /Applications.
